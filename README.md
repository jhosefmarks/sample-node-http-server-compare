# Sample for compare node`s servers.

## Http Server

### Run server

`node index.js`

### Open your favorite browser

- http://localhost:3000
- http://localhost:3000/profile
- http://localhost:3000/invalid

## Restify Http Server

### Run server

`node index-restify.js`

### Open your favorite browser

- http://localhost:3001
- http://localhost:3001/profile
- http://localhost:3001/invalid
